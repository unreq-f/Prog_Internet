Програма призначення для заповнення відомості відвідування занять студентами
Інформація про дії користувача логуються та відправляються на електронну пошту користувача
Для вказання пошти відправника та отримувача використовується .env файл
Використовується пароль застосунка замість справжнього пароля


pip install python-dotenv
